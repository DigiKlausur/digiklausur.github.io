Teacher Mode¶

In teacher mode all grading extensions (formgrader etc) are enabled.

Activate Teacher Mode¶

To activate teacher mode execute the following in a terminal:

python -m e2xgrader activate teacher --sys-prefix

Don’t forget to edit the nbgrader_config.py (see E2xgrader Configuration).
